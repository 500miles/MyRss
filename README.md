# MyRss
My Rss by Nodejs

> 该程序需要依赖[node](http://nodejs.org/)、[bower](http://bower.io/) 环境支持.

### 运行程序
1. 在项目根目录运行 `npm install` 命令，静待安装完成；
2. 进入 `public` 目录，执行 `bower install` 命令安装所依赖的前端框架;
3. 在项目根目录执行 `npm start` 启动程序即可；
